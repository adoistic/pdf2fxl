Step 02
i

Read the page, keep the geometry

Mistral OCR returns every text block with its bounding box. Thothica trusts what a block is, a heading or not, and keeps exactly where it sits.

ii

Measure the true type size

For each block, the ink rows give a line pitch. That is the real printed size, so a one-word heading in a wide box measures large, as it should.

iii

Recover the hierarchy, book-wide

Body size is the peak of the character-mass histogram. Larger clusters, reconciled with section numbering like 1.2.1, resolve to consistent levels across the whole book.

iv

Set every figure in proportion

Each figure and table keeps its share of the column, so a half-column figure stays half-column and a full plate stays full width.
