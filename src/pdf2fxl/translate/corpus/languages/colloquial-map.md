# South Asian Colloquial Language Map

When a South Asian language is the target, you can render it in an everyday/colloquial
spoken variant (with an explicit script instruction) instead of a stiff/literary register.
The original system applied this mapping unconditionally; this skill applies it
**stance-aware**:

- **Commercial stance** (everyday, marketing, journalism, digital, etc.) → apply the
  colloquial mapping by default, so output reads like natural spoken language.
- **Literary stance** (poetry, scripture, philosophy, classical/historical, literary
  fiction) → do NOT force the colloquial register; preserve the language's fuller
  literary register. Apply a mapping only if the user explicitly asks for a spoken/
  colloquial voice.

In all cases, honor the explicit **script** instruction in the "Render as" column.

Each language also has a `Simple <Language>` variant that adds "with not very
difficult words" to the same rendering instruction.
