# Critical Grammar Concepts for Kannada (ಕನ್ನಡ) Translation

## 1. Verbal Participles & Chains - **CORE SYNTAX**

Kannada chains actions using participles. Never use "and" to join two actions by the same subject.
- **English:** "He sat and read."
- **✓ Natural:** "**ಅವನು ಕುಳಿತು ಓದಿದನು**" (Avanu **kulithu** odhidhanu).
- **Kulithu** = Having sat.

---

## 2. Agglutination & Bridging Sounds

Suffixes merge with nouns using bridge letters (v/y/d).
- **Rule:** Ends in 'i/e' -> use **'y'**. (Mane + alli -> **Maneyalli**).
- **Rule:** Ends in 'a/u/o' -> use **'v'**. (Guru + annu -> **Guruvannu**).

---

## 3. Linguistic Decision Protocol: Kannada

When translating into Kannada, follow this specific thought sequence:

1. **Participle Chaining:** Are you joining two actions with 'mattu' (and)? If yes, convert the first action into a **Verbal Participle** (e.g., bandhu, hogo).
2. **Agglutination Buffers:** When attaching a case marker, did you use the correct bridge sound (**v, y, or d**)?
3. **Retroflex L Check:** Check for **ಳ** (Lla). Is the word 'Male' (Rain)? Use ಳ. Is it 'Maale' (Garland)? Use regular ಲ.
4. **Subject Case:** For internal states (feelings, health), use the **Dative Subject** (Nanage - "To me").
5. **Article Stripping:** Remove 'Ondu' (a/one) unless count emphasis is strictly required.

## Few-Shot Examples (Context-Aware)

- **English:** "I am cold."
